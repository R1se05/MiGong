**🧩 迷宫生成与求解系统**

一个使用Python实现的迷宫生成与求解系统，支持多种算法生成迷宫，并提供BFS求解、性能分析、文件保存等完整功能。

**✨ 功能特性**

**🏗️ 迷宫生成算法**

DFS回溯算法：递归深度优先，生成长通路迷宫

Kruskal算法：基于并查集，生成均匀分布迷宫

两种算法对比：支持性能比较和特点分析

**🖥️ 用户界面**

交互式控制台：友好的菜单驱动界面

ASCII可视化：纯字符图形显示迷宫

实时信息：显示算法类型、生成时间、尺寸等信息

**📊 性能分析**

算法效率测试：对比DFS和Kruskal算法性能

速度比计算：量化两种算法的效率差异

多尺寸测试：支持不同尺寸的性能对比

**💾 数据管理**

文件保存/加载：支持JSON格式的迷宫保存

状态恢复：完整恢复迷宫状态（包括路径）

连通性检查：验证迷宫是否有解

**🚀 快速开始**

环境要求

无需额外依赖库
